**Card** — neutral layered surface for grouping content. Sharp Carbon corners; add `accent` for a left status bar, `interactive` for clickable lists.

```jsx
<Card accent="var(--blue-60)">
  <h3>Authentication errors</h3>
  <p>12 related questions</p>
</Card>
```

Props: `padding`, `accent` (left bar color), `interactive` (hover lift), `selected` (blue border).
